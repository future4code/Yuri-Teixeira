Projetos

Pine Licence
Descrição

Software desenvolvido para gerenciar controle de licença de uso de outros softwares. Baseado na arquitetura MD5, modificada e personalizada para que seja uma criptografia bi-direcional. Desenvolvido utilizando C# e criptografia MD5.

Lua
Descrição

Software desenvolvido para controlar o gerenciamento de atendimento ao cliente. Proporcionando maior agilidade e organização, o Lua controla tarefas do dia-dia gerenciando clientes, anexos, categoria de atendimento e status. Desenvolvido utilizando C# e banco de dados MySQL.

MeritoADV
Descrição

Software desenvolvido para gestão de escritórios de advocacia, conta com uma carteira de clientes, processos, audiências, prazos, tarefas do dia-dia, controle de anexos, controle financeiro de contas a receber e a pagar. Desenvolvido utilizando C# e banco de dados MySQL.
